# Emoji

Use **Unicode emoji** throughout. They render in any Markdown previewer, editor, and platform — not just GitHub. Avoid GitHub shortcodes (`:name:` syntax): they only work on GitHub and a few other platforms.

---

## Section headers

```markdown
## 🔗 Más contenidos de SIMANFOR en GitHub
## 🎬 Vídeos introductorios al uso de SIMANFOR
## 💾 Repositorios de trabajos donde se ha utilizado SIMANFOR
## ✏️ Cómo citar SIMANFOR
## 💻 Lista de contribuidores
## 📧 Contacto
## ℹ️ Más información
```

🔨  — project group

---

## Usage rules

- Copy emoji directly from this file — do not type shortcodes.
- One space between combined prefixes: `💾 📚` (not `💾📚`).
- Emoji in headings go after the `##` marker and before the title text: `## 🔗 Title`.
